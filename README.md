# Thesis Systems

A web-based Graduate Thesis Management System developed using modern web technologies. This application allows users to search, view, and manage academic theses and authors efficiently.

## Features

* **Advanced Search:** Filter theses by title, author, type, institute, and year.
* **Thesis Management:** Add new theses, view details, and delete records.
* **Author Management:** Add new researchers/authors to the system.
* **System Architecture View:** A dedicated page explaining the frontend-backend-database workflow.
* **Responsive Design:** Built with React Bootstrap for a seamless experience on all devices.

## Tech Stack

* **Frontend:** React.js, React Bootstrap, Axios
* **Backend:** Node.js, Express.js
* **Database:** PostgreSQL

## Installation & Setup

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/Taylan361/db-final.git
    ```

2.  **Setup Backend**
    ```bash
    
    npm install
    node index.js
    ```
    *Ensure your PostgreSQL database is running and credentials are configured in `db.js`.*

3.  **Setup Frontend**
    ```bash
    
    npm install
    npm start
    ```

The application will run at `http://localhost:3000`.
